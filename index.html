📊 What Each Metric Means
📦 Output
Tasks processed vs daily target. Shows how productive you were each day.
Score = Actual ÷ Target × 100%
🎯 FPY (First Pass Yield)
Quality metric — how many tasks passed on the first submission without rejection. Higher = fewer errors.
Score = (Submitted − Rejected) ÷ Submitted × 100%
📞 Call Answer Rate
How many calls you answered out of total calls received. Minimum target is 95%.
Score = Answered ÷ Received × 100%
📅 Attendance
Days present out of required working days. Holidays are excluded unless you were required to work.
Score = Days Present ÷ Working Days × 100%
✅ Supervisor Tasks
Tasks assigned by your supervisor that you completed.
Score = Completed ÷ Assigned × 100%
📅 View Periods
Daily
Score for a specific day. Use the day pills (Mon / Tue / Wed...) to select which day to view.
WTD
Week-to-date. Accumulates from Monday through today of the selected week.
MTD
Month-to-date. Combines all weeks within the current calendar month.
QTD
Quarter-to-date. Shows your progress toward the full quarter goal. Q2 covers WW14 through WW26.
💡 Tips
•
Filter by function: Use the function buttons (Funding, Licensing, etc.) to view only your team.
•
Sort columns: Click any column header (Output, Attendance, Overall) to sort the leaderboard.
•
Trainees: Ranked separately at the bottom. Scores start from their onboarding date.
•
Green rows: Top Performers with ≥90% overall score.
•
Off metrics: Metrics showing "off" are not included in your overall score for that function.
🛠️ Admin Guide — How to Update the Dashboard
For supervisor/admin use only · PH SPWR Direct Ops v2.6
📋 Weekly Update Workflow
1
Open Dashboard — Go to the editor URL (without ?mode=view). Dashboard auto-restores your last session.
2
Select the Work Week — Use ◀ ▶ or the WW dropdown to navigate to the current work week.
3
Enter Daily Output — Data Entry → select Function tab → enter Target and Actual per day (Mon–Fri).
4
Mark Attendance — Check ☑ Present for each day. Check 🟠 H for holidays (not required = excluded from score). If both H and ☑ are checked = employee was required to work on a holiday and was present.
5
Enter FPY — Top row = Submissions, Bottom row = Rejections per day.
6
Enter Call Answer Rate — Top row = Answered, Bottom row = Received per day. Warning appears if Answered > Received.
7
Enter Supervisor Tasks — Top row = Assigned, Bottom row = Completed per day.
8
Update Leaderboard — Click Update Leaderboard ↗ to push all entries and see the ranked view.
9
Export Data — Click ⬇ Export All Weeks → saves SPWR-Leaderboard-Data.json.
10
Upload to GitHub — Go to github.com/spwr-ph-direct-ops/leaderboard → click SPWR-Leaderboard-Data.json → three dots → Upload/Replace → commit.
11
Verify — Open the viewer URL in incognito to confirm team sees updated scores.

👥 Roster Management
•
Add Coordinator: Roster tab → + Add Coordinator → enter Code, Name, Function, Role. For trainees set Start WW and Start Day.
•
Hide Coordinator: Roster tab → click Hide. Removes from leaderboard but preserves all historical data. Click ↩ Restore to reactivate.
•
Edit Details: Roster tab → edit Name, Function, Role, Target, Start WW/Day inline. Changes reflect immediately across all tabs.
•
Trainee start dates: Set Start WW and Start Day in Roster. Trainees only appear from their start week onwards.
